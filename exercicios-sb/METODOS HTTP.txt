METODOS HTTP (ELES NAO EXECUTAM NADA NO APP, É MAIS POR SEMANTICA NA APLICACAO):

REQ GET = SERVE PARA OBTER/LER INFOS DO SERVIDOR! (A RESP VEM NO CORPO (BODY)).
REQ POST = SERVE PARA INSERIR NOVOS DADOS NO SERVIDOR! Ex: CADASTRAR ALGO..., FORMULARIOS

REQ HEAD = SERVE PARA OBTER/LER INFOS DO SERVIDOR, PORÉM SEM O CONTEUDO!

QUANDO QUERO ALTERAR DADOS NO SERVIDOR: REQ PUT OU REQ PATCH

REQ PUT (MAIS USADO) = QUANDO QUERO ALTERAR TOTALMENTE (QUERO ALT TDS OS ATRIB DO OBJ)
REQ PATCH = QUANDO QUERO ALTERAR PARCIALMENTE (QUERO ALT ESPEC UM ATRIB DO OBJ)

REQ DELETE = SERVE PARA REALIZAR UMA EXCLUSAO.

REQ OPTIONS = SERVE PARA RETORNAR QUAIS METODOS HTTP O TAL MÉTODO SUPORTA!
